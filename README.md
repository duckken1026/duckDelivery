# duckDelivery

此專案主要是利用AR kit 和 coreML 去新增訂餐網站的功能。  
AR kit(擴增實境) : 購物系統可以用AR看到餐點放在桌上的樣子。  
coreML(機器學習) : 上傳圖片可以判斷使用者需要購買什麼樣的產品，偵測人臉的表情來推薦商品。
